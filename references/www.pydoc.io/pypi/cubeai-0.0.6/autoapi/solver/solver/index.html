solve(cube, heuristic=None, verbose=False)¶

solves a given cube instance :param cube: cubeai.cube.cube.Cube instance :param heuristic: a heuristic function for A* search.

takes a cube and problem instances as input and returns a number. by default, uses the null heuristic (0 for every state)
Parameters:verbose – if True, prints some information about the search
Returns:a list of actions that solves the cube

returns a sequence of random actions on the given cube (does not apply the actions to the cube) :param cube: Cube object :param num_actions: an integer >= 1 :param quarter_metric: whether to use the quarter-turn metric or not :return: a sequence of `num_actions’ actions
